#database/analytics - Amazon Athena is a serverless query service to perform analytics against S3 objects. - Uses standard SQL to query the files - Supports CSV, JSON, ORC, Avro, and Parquet (Presto) - - Pricing is $5.00 per TB of data scanned. - Use compressed or columnar data for cost savings - Use cases: Business intelligence, analytics, reporting, analyze and query VPC flow logs, ELB logs, CloudTrail trails etc #### Exam tip: "Analyse data in S3 using serverless SQL" -> Athena
