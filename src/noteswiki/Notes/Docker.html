#compute - Docker is a Software Development platform to deploy apps - Apps are packaged in containers that can be run on any OS - Apps run the same, regardless of where they're run - Any machine - No Compatibility issues - Predicable behaviour - less work - easier to maintain and deploy - works with any language, any OS, any Technology. - Scale containers up and down very quickly ### Docker on an OS ### Where are docker images Stored - Docker images are stored in docker Repositories. - Public: Docker Hub https://hub.docker.com/ - Find base images for many techs or OSs - Ubuntu, MySQL, node etc - Private: Amazon ECR (Elastic Container Registry) ### Docker vs VM - Docker is sort of a virtualization tech. (not quite) - Resources are shared with the host. So many containers can be run on a server - VM vs Docker diagram - - Docker is lighter weight than a VM
