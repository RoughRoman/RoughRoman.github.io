#database/analytics - Redshift is based on PostgreSQL, but is not used for OLTP (Online Transaction Processing system) - Its OLAP- Online analytical processing (for analytics and data warehousing. ) - Load data once every hour, not every second. - 10x better performance than other data warehouses and scaled to Petabytes of data - Uses a columnar storage model of data(Instead of rows.) - Massively parallel query execution (MPP), highly available - pay as you go based on the instances provisioned. - has a SQL interface for performing the queries - BI tools such as AWS QuickSight or Tableau intergrated with it.
