#database/analytics - Like how Aurora was an AWS cloud native solution for PostgreSQL and MySQL - DocumentDB is the same for MongoDB (NoSQL) - _Context: MongoDB is used to store, Query, and index JSON data._ - Similar deployment concept as Aurora - Fully managed, highly available with replication across 3 AZs - DocumentDB storage automatically grows in increments of 10GB, up to 64TB - Automatically scales to workloads with millions of requests per second.
